เริ่มต้นสร้างแชนแนล


ท่านสามารถสร้างแชนแนลของท่านเพื่อสร้าง Token Key และเริ่มต้นใช้บริการ APIs ที่ทางสำนักงานศูนย์เทคโนโลยีสารสนเทศและการสื่อสารให้บริการได้เพียงกด สร้างแชนแนล



เริ่มใช้งาน


1.คลิกปุ่ม Go To Console



2.กรอก Username Password บัญชีที่ทาง คณะวิศวกรรมศาสตร์ ออกให้เท่านั้น



3.อ่านนโยบายความเป็นส่วนตัวแล้วกด Agree



4.คลิกเลือกเมนู "ตั้งค่า"



5.ไปที่เมนู "ตั้งค่าความปลอดภัยแบบสองขั้นตอน" คลิกปุ่ม "Enable Two-Factor Authentication"



6.คลิกปุ่ม Enable



7.ระบบจะแสดง QR Code ผ่าน Application "Google Authenticator"



8.บน Application จะปรากฏเลข 6 หลัก ซึ่งสามารถนำมาใช้ยังระบบเพื่อ เพิ่ม แก้ไข และ ลบข้อมูลได้
